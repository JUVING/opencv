
## *함수*

| 함수         | 기능                                   | 사용법                                       |
| ------------ | -------------------------------------- | -------------------------------------------- |
| imread()     | 영상 호출 (BMP, JPG, TIF, PNG )        | imread("영상 이름")                          |
| namewindow() | 화면에 새로운 창 생성                  | namewindow("창 이름")                        |
| imshow()     | 화면 창에 영상 출력                    | imshow("출력할 창 이름", 영상을 저장한 변수) |
| waitkey()    | 사용자가 키보드 누르기 전 까지 출력    | waitkey()                                    |
| imwrite()    | Mat에 저장된 영상 데이터를 파일로 저장 |                                              |
